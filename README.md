自己的遊戲自己寫 Part 0 - 水滴遊戲
===================================

這是 libGDX 教學系列影片「自己的遊戲自己寫」的第一個小遊戲。這個教學影片系列的目標，是實作 20 Games Challenge 裡的 2D 小遊戲，並將過程記錄下來。Part 0 的部份是 libGDX 的基礎教學，由官方文件 A Simple Game 修改而來。

EP08
---------------

介紹如何在 libGDX 中實作簡單的碰撞偵測，以及背景音樂與音效的撥放。
